Χρησιμοποιώντας Πρόσθετες Λειτουργίες¶

Τα Plugins στο QGIS προσθέσουν χρήσιμες λειτουργίες στο λογισμικό.Τα Plugins γράφτηκαν από προγραμματιστές του QGIS και άλλους ανεξάρτητους χρήστες που θέλουν να επεκτείνουν την βασική λειτουργικότητα του λογισμικού. Αυτά τα Plugins διατίθενται σε QGIS για όλους τους χρήστες.

Επισκόπηση του έργου¶

Σε αυτό το tutorial, θα μάθετε πώς μπορείτε να ενεργοποιήσετε τα Core Plugins καθώς και να κατεβάσετε και να εγκαταστήσετε τα External Plugins. Θα μάθετε επίσης πώς να εντοπίσετε το plugin από το μενού του QGIS από τη στιγμή που έχουν εγκατασταθεί.

Διαδικασία¶

Βασικά Plugins¶

Τα βασικά plugins είναι ήδη μέρος της τυπικής εγκατάστασης του QGIS. Για να τα χρησιμοποιήσετε , το μόνο που χρειάζεται είναι να τους δοθεί η δυνατότητα.

  1. Ανοίξτε QGIS. Κάντε κλικ στο: Επιλογή μενού:Plugins –> Manage and Install Plugins.... για να ανοίξετε το:guilabel:Plugin Manager διαλόγου.

  1. Ακόμα κι αν αυτή είναι η πρώτη φορά που χρησιμοποιείτε το QGIS, θα δείτε πολλά plugins που αναφέρονται στο πλαίσιο του:guilabel:Installed tab. Αυτό συμβαίνει επειδή είναι Core Plugins και είχαν εγκατασταθεί κατά την εγκατάσταση του QGIS.

  1. Ας ενεργοποιήσετε ένα από τα plugins. Ελέγξτε στο κουτάκι δίπλα στο Spatial Query Plugin. Αυτό θα ενεργοποιήσει το plugin και θα είστε σε θέση να το χρησιμοποιήσετε. Ένα πράγμα που πρέπει να σημειωθεί είναι ότι τα πρόσθετα έχουν τη δυνατότητα να εισάγουν τα στοιχεία μενού σε διάφορες θέσεις και να δημιουργούν νέες επιτροπές και γραμμές εργαλείων. Μερικές φορές είναι δύσκολο να γνωρίζει κανείς πώς να βρείτε τα πρόσφατα ενεργοποιημένα εργαλεία. Ένα στοιχείο είναι να κοιτάξουμε στην περιγραφή του plugin. Εδώ η περιγραφή λέει Category: Vector. Αυτό δείχνει ότι το plugin θα βρεθεί κάτω από το:guilabel:Vector`μόλις ενεργοποιηθεί το μενού. Κάντε κλικ :guilabel:`Close.

  1. Τώρα που η Spatial Query Plugin είναι ενεργοποιημένη, μπορείτε να πάτε στο :Επιλογή μενού:Vector –> Spatial Query για να χρησιμοποιήσετε τη λειτουργικότητα που προστίθεται από το plugin.

Εξωτερικά Plugins¶

Tα εξωτερικά plugins είναι διαθέσιμα στο QGIS Plugins Repository και πρέπει να εγκατασταθούν από τους χρήστες πριν από τη χρήση τους. Ένας εύκολος τρόπος για να περιηγηθείτε και να εγκαταστήσετε αυτά τα plugins είναι με τη χρήση του εργαλείου Plugin Manager.

  1. Ανοίξτε QGIS. Κάντε κλικ στο: Επιλογή μενού:Plugins –> Manage and Install Plugins.... για να ανοίξετε το:guilabel:Plugin Manager διαλόγου.

  1. Κάντε κλικ στο Get more tab. Εδώ θα δείτε μια λίστα των plugins που αναφέρονται.

  1. Γι αυτό το tutorial, ας βρούμε και να εγκαταστήσουμε ένα plugin που ονομάζεται ‘QuickQKT’. Μόλις αρχίσετε να πληκτρολογείτε*qui* στο κουτί search , θα δείτε τα αποτελέσματα της αναζήτησης παρακάτω. Κάντε κλικ στο:guilabel:QuickWKT. Επειτα, κάντε κλικ στο κουμπί Install plugin για να το εγκαταστήσετε.

  1. Όταν το plugin έχει κατεβεί και εγκατασταθεί, θα δείτε ένα παράθυρο διαλόγου επιβεβαίωσης.

  1. Αν έχετε παρατηρήσει, δεν υπήρξε καμία αναφορά της κατηγορίας plugin στην περιγραφή. Αυτό καθιστά δύσκολο να προσδιοριστεί πώς να αποκτήσετε πρόσβαση στο πρόσφατα εγκατεστημένο plugin. Τα περισσότερα εξωτερικά πρόσθετα εγκαθίστανται κάτω από το ίδιο μενού:guilabel:Plugins στο QGIS. Κάντε κλικ στο: Επιλογή μενού:Plugins –> QuickWKT και θα δείτε το μόλις εγκατεστημένο plugin. Συνήθως, τα εξωτερικά plugins έχουν επίσης κουμπί εγκαταστασης στο Plugins εργαλείων, επίσης. Μπορείτε επίσης να χρησιμοποιήσετε αυτό το κουμπί για να αποκτήσετε πρόσβαση στο πρόγραμμα.

Πειραματικά Plugins¶

Τώρα ξέρετε πώς να εγκαταστήσετε και να βρει μια*External Plugin* στο QGIS. Ας διερευνήσουμε κάποιες επιλογές για προχωρημένους. Μερικές φορές ψάχνετε για ένα συγκεκριμένο πρόγραμμα, αλλά δεν μπορείτε να το βρείτε στο:guilabel:Get more tab. Είναι ίσως επειδή το plugin έχει επισημανθεί Experimental. Εδώ είναι πώς να εγκαταστήσετε experimental plugins.

  1. Ανοίξτε Plugin Manager by :Επιλογή μενού:Plugins –> Manage and Install Plugins.... Κάντε κλικ στο Settings tab.Θα δείτε μια επιλογή που ονομάζεται Show also experimental plugins. Κάντε κλικ στο πλαίσιο ελέγχου δίπλα σε αυτό, για να το ενεργοποιήσετε.

  1. Θα δείτε μια νέα καρτέλα που ονομάζεται New. Τα πρόσφατα ενεργοποιήμένα πειραματικά plugins θα εμφανιστούν εδώ.

Note

Η καρτέλα New θα εμφανιστεί μόνο προσωρινά τη στιγμή που θα ενεργοποιήσετε τα πειραματικά plugins. Την επόμενη φορά που θα ανοίξετε τα πειραματικά plugins Plugin Manager, θα δείχνουν παράλληλα με την τακτική των plugins στη Get more tab.

  1. Ας εγκαταστήσουμε ένα plugin που ονομάζεται TimeManager. Κάντε κλικ στο όνομα του plugin και, στη συνέχεια, κάντε κλικ: Install.

  1. Τώρα, όταν θα επιστρέψετε στο κύριο παράθυρο του QGIS, θα δείτε ένα νέο Panel στο κάτω μέρος του πίνακα. Αυτός ο πίνακας έχει δημιουργηθεί από το plugin TimeManager. Αυτός είναι ένας ακόμη τρόπος των plugins για να προσθέσετε λειτουργικότητα στο περιβάλλον εργασίας του χρήστη.

  1. Μπορείτε να ενεργοποιήσετε / απενεργοποιήσετε αυτό το πάνελ από: επιλογή μενού:View –> Panels –> Time Manager.
