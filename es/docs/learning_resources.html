QGIS Learning Resources¶

The QGIS project has a vibrant community that has created a lot of good documentation and resources that one can use to learn the software as well as GIS techniques. Here are a few selected external resources.

Official Documentation¶

  • QGIS 2.0 Documentation
  • A Gentle Introduction to GIS
  • The QGIS Training Manual

Training Manual¶

  • GIS Practicum - Introduction to GIS Using Open Source Software by Frank Donnelly, Baruch College CUNY

Books¶

  • Graser, Anita Learning QGIS 2.0, Packt Publishing, 2013
  • Kurt Menke, GISP, Dr. Richard Smith Jr., GISP, Dr. Luigi Pirelli, Dr. John Van Hoesen, GISP Mastering QGIS, Packt Publishing, 2015
  • Sherman, Gary The PyQGIS Programmer’s Guide, Locate Press, 2014
