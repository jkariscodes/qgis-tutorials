Estilos y Análisis Básicos Ráster¶

Advertencia

Este tutorial está ahora obsoleto. Una versión nueva y actualizada está disponible en Estilos y Análisis Básicos Ráster (QGIS3)

Un montón de observaciones e investigación científica producen conjuntos de datos ráster. Los rásters son esencialmente cuadrículas de píxeles que tienen un valor específico asignado a ellos. Haciendo operaciones matemáticas en esos valores, se puede hacer algunos análisis interesantes. QGIS tiene algunas capacidades básicas de análisis incorporadas mediante la Calculadora Ráster. En este tutorial, exploraremos lo básico en el uso de la Calculadora Ráster y opciones disponibles para dar estilo a rásters.

Vista general de la tarea¶

Usaremos datos cuadrícula de densidad de población para encontrar y visualizar áreas del mundo que han visto cambio dramático en densidad de población entre el año 1990 y 2000.

Otras habilidades que aprenderás¶

  • Seleccionar y cargar múltiples conjuntos de datos en un solo paso en QGIS.

Obtener los datos¶

Usaremos el conjunto de datos Gridded Population of the World (GPW) v3 de la Universidad Columbia. Específicamente, necesitaremos la Cuadrícula de Densidad de Población para todo el globo en formato ASCII y para el año 1990 y 2000.

Aquí va como buscar y descargar los datos relevantes.

  1. Vaya a la página de descarga Population Density Grid, v3. Seleccione para Atributos de Datos .ascii format, 1° resolution y 1990 year. Clic en Descargar. Para este punto, puede crear una cuenta gratuita e ingresar, o usar el botón Descarga Invitado abajo para descargar inmediátamente los datos. Repita el proceso para el 2000 year

Ahora tendrá 2 archivos zip descargados.

Para su comodidad, puede descargar directamente una copia de los conjuntos de datos de los enlaces abajo:

gl_gpwv3_pdens_90_ascii_one.zip

gl_gpwv3_pdens_00_ascii_one.zip

Fuente de Datos [GPW3]

Procedmiento¶

  1. Abra QGIS y vaya a Capas ‣ Añadir capa Ráster...

  1. Localice los archivos zip descargados. Mantenga presionada la tecla Ctrl y clic en ambos archivos zip para seleccionarlos. De esta manera es capa de cargar ambos archivos en un solo paso.

  1. Cada archivo zip contiene 2 archivos grid. la a en el nombre de archivo sugiere que los conteos de población fueron ajustados para coincidir con los totales de las NN.UU. Usaremos las cuadrículas ajustadas para este tutorial. Seleccione glds00ag60.asc como la capa a añadir. Clic en Aceptar.

  1. La capa no tiene un SRC definido, y debido a que las cuadrículas están en lat/lon, escoja EPSG:4326 como el sistema de referencia de coordenadas.

  1. Debido a que seleccionamos ambos archivos zip, veremos diálogos similares una vez más. Repita el proceso y seleccione la cuadrícula glds90ag60.asc como la capa a añadir.

  1. Una vez más, escoja EPSG:4326 como el SRC.

  1. Ahora verá ambos ráster cargados en QGIS. El ráster es representado como una escala de grises, donde los píxeles más oscuros indican valores más bajos y los píxeles más claros indican valores más altos.

  1. Cada píxel en el ráster tiene un valor asignado. Este valor es la densidad de población para esa cuadrícula. Clic en el botón Identificar Entidades para seleccionar la herramienta y clic en cualquier parte del ráster para ver el valor de ese píxel.

  1. Para visualizar mejor el patrón de densidad de población, necesitaríamos darle estilo. Clic-derecho en el nombre de la capa y seleccione Propiedades. También puede hacer doble-clic en el nombre de la capa en la TDC para traer el diálogo Propiedades de Capa.

  1. Bajo la pestaña Estilo, cambie el Tipo de representador a Pseudocolor banda única. A continuación, clic en Clasificar bajo Generar un nuevo mapa de color. Verá 5 nuevos valores de color creados. Clic en Aceptar.

  1. De vuelta en la Pantalla QGIS, verá una representación como mapa de calor del ráster. Repita también el mismo proceso para el otro ráster.

  1. Para nuestro análisis, nos gustaría encontrar áreas con los mayores cambios de población entre 1990 y 2000. La manera de lograr esto es encontrando la diferencia entre los valores de píxeles grid en ambas capas. Seleccione Ráster ‣ Calculadora ráster.

  1. In the Raster bands section, you can select the layer by double-clicking on them. The bands are named after the raster name followed by @ and band number. Since each of our rasters have only 1 band, you will see only 1 entry per raster. The raster calculator can apply mathematical operations on the raster pixels. In this case we want to enter a simple formula to subtract the 1990 population density from 2000. Enter glds00ag60@1 - glds90ag60@1 as the formula. Name your output layer as pop_density_change_2000_1990.tif and check the box next to Add result to project. Click OK.

  1. Una vez está completa la operación, verá que se carga la nueva capa en QGIS.

  1. This grayscale visualization is useful, but we can create a much more informative output. Right-click on the pop_density_change_2000_1990 layer and select Properties.
  1. We want to style the layer so pixel values in certain ranges get the same color. Before we dive in to that, go to the Metadata tab and look at the properties of the raster. Note the minimum and maximum values of this layer.
  1. Now go to the Style tab. Select Singleband pseudocolor as the Render type under Band Rendering. Set the Color interpolation to Discrete. Click the Add entry button 4 times to create 4 unique classes. Click on an entry to change the values. The way color map works is that all values lower than the value entered will be given the color of that entry. Since the minmum value in our raster is just above -2000, we choose -2000 as the first entry. This will be for the No Data values. Enter the values and Labels for other entries as below and click OK.
  1. Now you will see a much more powerful visualization where you can see areas which has seen positive and negative population density changes. Click on Zoom In button and draw a rectangle around Europe to explore the region in more detail.
  1. Select the Identify tool and click on the Red and Blue regions to verify that your styling rules worked as intended.
  1. Now let’s take this analysis one-step further and find areas with only negative population density change. Open Raster ‣ Raster calculator.
  1. Enter the expression as shown below What this expression will do is set the value of the pixel to 1 is if matches the expression and 0 if it doesn’t. So we will get a raster with pixel value of 1 where there was negative change and 0 where there wasn’t. Name the output layer as negative_pop_change_2000_1990 and check the box next to Add result to project. Click OK.
pop_density_change_2000_1990@1 < -10
  1. Once the new layer is loaded, right-click on it and select Properties. In the Transparency tab, add 0 as the Additional no data value. This setting will make the pixels with 0 values also transparent. Click OK.
  1. Now you will see the areas of negative population density change as gray pixels.
